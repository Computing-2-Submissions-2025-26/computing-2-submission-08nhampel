(static) chord(row, col, game) → {Minesweeper.Game}

Description:

  • Chords a revealed numbered cell, auto-revealing its remaining neighbours. Chording is an advanced move: if the player has placed exactly as many flags around a numbered cell as the cell's clue indicates, all unflagged neighbours are revealed at once. This speeds up play once the player is confident in their flag placement.

    If any flag is misplaced (on a safe cell rather than a mine), chording may reveal a mine and lose the game.

    The game is unchanged if: the game is over, the cell is not revealed, the cell's clue is 0, or the number of surrounding flags does not match the clue.

(static) reveal(row, col, game) → {Minesweeper.Game}

Description:

  • Reveals the cell at the given position. This is the primary action a player takes on their turn.

    On the first move of the game, mines are placed automatically to guarantee the chosen cell and its neighbours are safe.

    If the revealed cell contains a mine, the game is lost and the mine is marked as revealed so the UI can show where it was.

    If the revealed cell has no mine neighbours (clue of 0), all connected safe cells are revealed automatically in a cascade — saving the player from clicking each empty cell individually.

    The game is unchanged if: the game is already over, the cell is already revealed or the cell is flagged.
